Theme SCSS Line

Import line-chart-theme or chart-theme into your styles.scss.
If you only import line-chart-theme you also need tooltip-chart-theme for the Hover tooltip if not already imported. 

@import '~dl-chart/lib/line-chart/line-chart.theme';

@function light-line-chart-theme()

predefined light Theme

@function dark-line-chart-theme()

predefined dark Theme

@function line-theme($line-color, $text-color, $raster-line-color, $selection-line-color)

custom Theme function

@mixin line-chart-theme($theme)

mixin to use a Theme
