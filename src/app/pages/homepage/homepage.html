dlChart

lightweight Angular Chart

npm install dl-chart
Pie/Donut Chart Bar Chart Line Chart Legend Chart Item Service Github

Fast loading

Fast loading time because of the small package size without the need of any additional dependencies.

TypeScript & Angular & SVG

Build with the goal for a nativ lightweight Angular Charts library in mind for Dashboards and other needs.

Flexible

Additional data can be set in the Chart values which will then be provided in the Click and Select events.

Legend module

Use the buildin Legends module or import the ChartItem service and create your own.
