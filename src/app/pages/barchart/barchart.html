Theme SCSS Bar

Import bar-chart-theme or chart-theme into your styles.scss.
If you only import bar-chart-theme you also need tooltip-chart-theme for the Hover tooltip if not already imported.

@function dark-bar-chart-theme()

@function bar-theme($line-color, $text-color, $ground-line-color, $selection-line-color, $select-stroke-width, $selection-stroke-dasharray)
